# [new term] Correction

*Paste into a **New term proposal** issue. Term file already drafted at `terms/correction.yaml` (status: draft).*

**Proposed term:** Correction

**Proposed definition:**

> A dispute outcome in which an adjudicator determines that the record's content was wrong from the start and restates it as the content the law recognises as having held all along. The party-incident-land triple is retained; the corrected version re-states the same valid-time interval and supersedes the erroneous version in transaction time only. The error may be trivial or substantive — a misspelt name and a lease recorded as 25 years that was granted for 99 are both corrections, because in each case the triple is intact and the true position always held; the register simply said otherwise. Severity determines which adjudicator is empowered, not which operation runs: a correction becomes a *Rectification* only when the triple itself is wrongly stated. Distinguished from *Amendment*, which is party-driven, changes the terms, and moves valid time.
